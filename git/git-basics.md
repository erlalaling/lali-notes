# Git Basics

## What is Git?

Git is a version control system that tracks changes in files and projects.

## Commands I Learned

### git init
Creates a new repository.

### git add .
Stages all changes.

### git commit -m "message"
Creates a snapshot of changes.

### git status
Shows the current state of the repository.
